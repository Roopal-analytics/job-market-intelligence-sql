# job-market-intelligence-sql
End-to-end SQL project analyzing job market trend, skills, salaries, and locations. 

## Overview
This project analyzes job market data to understand skill demand, salary trends,
and city-wise hiring patterns. The goal is to help job seekers and organizations
make informed decisions using data.

## Business Problem
Job seekers often learn skills without knowing actual market demand.
This system uses structured SQL analysis to identify:
- In-demand skills
- High-paying skill combinations
- Fresher-friendly cities
- Hiring trends over time

## Data Design
The database is designed using a normalized schema with multiple related tables
to reflect real-world job listing structures.
